Algoritmo SIMPLES de Criptografia, desenvolvido em JS

Delorean em Ação!


Como Funciona

O algoritmo Cryptography Delorean, é desenvolvido em JavaScript puro. e pode ser facilmente implementado.

A lógica por trás do algoritmo

Assim como na famosa Cifra de César que hoje parece algo extremamente intuitivo, mas na época foi a responsável pela dor de cabeça de muitos soldados, nossa criptografia terá uma implementação fácil . Por isso não se engane com a facilidade da implementação, o segredo por trás é desenvolver algo único. Quanto mais exclusivo forem os métodos por trás de uma criptografia, mas difícil será burla-la, então sinta-se à vontade para usar esse algoritmo como base e criar um modelo seu totalmente novo.


Passo 1

O primeiro passo é simples, nessa etapa só pegamos a informação que sera criptografada e separamos os caracteres:

A função "string.split()" nos permite receber uma string e convertela para uma array, separado os caracteres extamente como queriamos

var string = 'Yoda';
var array = string.split('');

Passo 2

Agora precisamos converter cada caractere para um valor numérico que o represente, para fazer isso usaremos o método charCodeAt(0)

O método charCodeAt(0) retorna um inteiro entre 0 e 65535 representando a unidade de código UTF-16. Percorrendo todas as posições da nossa Array podemos converter toda nossa string para o modelo UTF-16

for(i =0 ; i != array.length;i++){
   array[i] = array[i].charCodeAt(0);
}

Passo 3

Agora temos os dados no formato UTF-16, só precisamos aplicar nossa formula para criamos nossa própria criptografia. E aqui está o segredo, vamos pegar o valor de caractere por caractere e aplicamos uma formula:

    Imagine que a letra [A] tenha um valor UTF-16 igual a 2. Poderíamos simplesmente aplicamos a formular matemática que quiséssemos para gerar um novo valor.

    Um exemplo simples seria [A] * 2, sabendo que A é equivalente a 2, após nosso calculo o valor seria 4, o que em UTF-16 geraria um caractere totalmente diferente. Já poderíamos considerar isso como uma criptografia, mas não seria difícil encontrar um padrão analisando uma quantidade adequada de casos.

    Para dificultamos as coisas, vamos finalmente aplicar o diferencial da nossa criptografia, o elemento mutável. O segredo por trás de tudo é usar o tempo como uma variável na hora de realizamos nossas operações. Daí que saiu o nome Delorean, (não podia perder a chance de fazer uma referência ao De Volta Para o Futuro).

    Com a função getHours(), podemos pegar a hora exata em que a nossa função foi executada. Nosso código agora ficará assim:

var time = new Date();

for(i =0 ; i != array.length;i++){
   array[i] = time.getHours() * array[i].charCodeAt(0);
}


    Usando a variável tempo, o mesmo "A", que foi representado como por um valor , terá um o valor totalmente diferente caso tenha sido gerado em outra hora. E a coisa pode ficar muito mais complexa, no exemplo usamos a hora, mas você ainda pode aplicar mais variáveis de tempo, como : getMinutes() & getSeconds().

    Pronto, Criptografia criada !
Passo 4

Agora você deve estar se perguntando como o recebedor dessa informação vai conseguir descriptografar e conseguir ter acesso aos dados.

    Para realizar a descriptografia precisamos fazer a engenharia inversa, e para realizar isso o receptor irar precisar do nosso valor chave, o que no nosso caso é a hora que a informação foi criptografada. então precisamos passa essa informação junto com o nosso valor, o que deve ficar simples assim:

var time = new Date();

for(i =0 ; i != array.length;i++){
   array[i] = time.getHours()+""+ time.getHours() * array[i].charCodeAt(0);
}


E chegamos ao fim, essa é a criptografia Delorian, espero que ela possa ser útil e que auxilie na criação de novas criptografias. O código completo com todos os detalhes da implementação está disponível no meu GitHub.
